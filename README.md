# React_Third_Project
About Country Explorer

Description:
This app allows users to explore countries using real API data.
You can search countries, filter by region, and add favorites.

How to run:
npm run dev

API Used:
https://restcountries.com/v3.1/all  
https://restcountries.com/v3.1/name/{name}  
https://restcountries.com/v3.1/region/{region}

Features:
- Search countries
- Filter by region
- Favorite countries 
- Loading & error handling
